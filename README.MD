# fast-command

> Customize your command (git command) by alias.

## Install

```bash
npm i fast-command -g
```

## Usage

### List all commands

```bash
fast-command -l
```

### Use command

```bash
fast-command -c <command-alias>
```

```bash
fast-command -c delete-all-branch
```

### Customize command

#### Add command

```bash
fast-command config add
```

#### delete command

```bash
fast-command config -d <command-alias>
```
